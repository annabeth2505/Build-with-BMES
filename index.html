What is Arduino?

Arduino is a small computer designed to interact with the physical world — to read buttons, light up LEDs, spin motors, beep at things. The chip is tiny, the code is short, and the rules are surprisingly forgiving.

Today you won't touch real hardware. You'll use Tinkercad, a free browser tool that simulates Arduino circuits visually. Wire it wrong? Nothing burns. Code a typo? Nothing breaks. It's the safest, most patient lab in the world.

How this works

  • Each project page lists the components you need to drag into Tinkercad
  • Hints walk you through the logic — not the answers
  • Try, fail, fix, repeat. That's the actual job of an engineer.
  • Stuck? Ask a peer or facilitator before peeking at hints
  • The last project has unlockable bonus levels — only revealed once you finish the base
One rule Read the hint before you copy code. Understanding beats finishing.
