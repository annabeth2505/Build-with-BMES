Project 01

Blink an LED.

One light. On. Off. On. Off. It sounds boring — but every Arduino project on Earth starts with this. By the end you'll know how to make a microcontroller actually do something in the real world.

Make an LED blink on and off

The light should turn on, wait a moment, turn off, wait again, and repeat — forever. You decide how fast.

Heads up: LEDs have a direction The longer leg is the positive side (anode). The short leg goes to ground. Flip it and nothing happens — no harm, just no light.
02

Wire it up

  • Drop the Arduino and breadboard onto the Tinkercad workspace
  • Connect the Arduino's 5V pin to the red rail of the breadboard, and GND to the blue rail
  • Plug the LED into the breadboard (long leg one row, short leg the next)
  • Connect the LED's long leg, through the resistor, to a digital pin on the Arduino — try pin 13
  • Connect the LED's short leg to the GND (blue) rail
03

Hints to write the code

Two functions, always. Every Arduino sketch has setup() — runs once when the board starts — and loop() — runs over and over forever. You'll need both.
Tell the pin which way it goes. Inside setup(), use pinMode(pinNumber, OUTPUT) to declare that the pin will be sending power out (to the LED), not reading something in.
Turn the pin on and off. digitalWrite(pinNumber, HIGH) sends power. digitalWrite(pinNumber, LOW) stops. Put both inside loop().
Wait between changes. Use delay(milliseconds) to pause. delay(1000) is one second. Without delays, the LED switches so fast it just looks dim.
Order matters. ON → wait → OFF → wait. Try blinking once per second first. Then try faster. Then SOS pattern. Make it yours.
04

Try these once it works

  • Make it blink twice as fast — what changes?
  • Make it blink in a heartbeat pattern: short-short-long pause
  • Add a second LED on a different pin, blinking out of sync
  • Spell SOS in Morse code (·· ·· ·· — — — ·· ·· ··)

Code cheat sheet

setup() vs loop()setup() runs once. loop() runs forever. Both are required.
Declare an outputpinMode(13, OUTPUT); goes in setup().
Turn on / offdigitalWrite(13, HIGH) = on. digitalWrite(13, LOW) = off.
Waitdelay(1000) pauses for 1 second (1000 ms).

Stuck? try this
